we-vue 是一套基于vue.js 的移动关组件库，结合 weui.css 样式库，封装了一系列组件，非常适于微信公众号等移动端开发。

## 安装

```shell
// npm
$ npm install we-vue -S

// yarn
$ yarn add we-vue -S
```

## 开发

1. 克隆代码

```shell
$ git clone https://github.com/tianyong90/we-vue.git
```

2. 安装依赖

```shell
$ cd we-vue && npm install
```

3. 运行本地服务并开发

```shell
$ npm run dev
```
